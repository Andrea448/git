GIT

Git je sustav za kontrolu verzija koji omogućava praćenje promjena u datotekama i suradnju među više ljudi na projektima. Svaka osoba ima svoju kopiju cjelokupno uključujući povijest svih promjena, što omogućava rad, kao i lako spajanje različitih verzija projekta.

  • Nastao je 2005. godine, razvio ga je Linus Torvalds
  • Git je dizajniran da bude vrlo brz, sa optimizacijom za rad sa velikim projektima.
  • Git olakšava kreiranje grana, što omogućava paralelni rad na različitim funkcionalnostima ili ispravkama. Kada se rad na grani završi, grana se može jednostavno spojiti sa glavnom granom, čime se omogućava upravljanje.
  • Git prati sve promjene u kodu kroz commits. Svaki commit sadrži informacije o tome šta je promjenjeno, ko je napravio promjene, kada su napravljene, i razlog za promjene.
  • Git koristi SHA-1 hash funkciju za identifikaciju svakog commit-a, što znači da je povijest projekta zaštićena od neovlašćenih promjena.
  • Git omogućava lako povezivanje sa udaljenim serverima (npr. GitHub, GitLab, Bitbucket)
  • Ukratko git je alat za kontrolu verzija koji omogućava brzu, sigurnu i suradnju na projektima, posebno kada više ljudi radi na istom kodu.
